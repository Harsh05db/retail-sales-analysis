# Retail Sales Analysis Using MySQL

## Project Overview

This project analyzes retail sales data from January to April 2025 using MySQL.

The objective is to identify:

* Revenue trends
* Product performance
* Category performance
* City-wise sales contribution
* Average order value
* Business insights for decision making

---

## Tools Used

* MySQL
* Excel
* GitHub

---

## Database Structure

| Column     | Description             |
| ---------- | ----------------------- |
| Order_ID   | Unique order identifier |
| Order_Date | Date of purchase        |
| Product    | Product sold            |
| Category   | Product category        |
| Quantity   | Units sold              |
| Unit_Price | Price per unit          |
| City       | Sales location          |
| Revenue    | Quantity × Unit Price   |

## Business Questions Answered

1. Total revenue generated
2. Monthly revenue comparison
3. Top-selling products
4. Revenue by category
5. Revenue by city
6. Average order value
7. Highest revenue orders
8. Revenue contribution analysis

---

## Key Insights

* Electronics generated the highest revenue.
* Laptops were the top revenue-generating product.
* Revenue varied across cities.
* High-revenue products were not always the highest-volume products.
* Average order value provided insights into customer spending behavior.
